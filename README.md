# PyShopBot - Бот который повысит Вашу прибыль
Бот написан на языке Python, использует напрямую API Vkontakte с использованием Threading.

## Функционал:
- Имеет 2 основные модели: `product` и `basket`.
  - С моделью `product` взаимодействует только администраторы бота. Там хранится информация о: наименовании товара, описание, цена, минимальное-максимальное количество товара которое можно взять и общая сумма покупок этого товара.
  - С моделью `basket` взаимодействует пользователь бота. Там хранится информация о: покупаетеле, какой товар был выбран, количество товара, статус товара, общая стоимость если товара больше 1 штуки.
